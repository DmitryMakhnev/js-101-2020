# Project with modules

## Requirements

- [Node.js](https://nodejs.org/en/)
(recommend to install with
[`nvm`](https://github.com/nvm-sh/nvm)
or [`nvm-windows`](https://github.com/coreybutler/nvm-windows))

## Install dependencies

```shell script
npm install
```

## Start development

```shell script
mon start
```
